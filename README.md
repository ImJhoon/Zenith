# 🏔️ Zenith

Zenith는 웹 기반 메모 애플리케이션입니다.

## 📖 이름의 기원

Zenith(정점)라는 이름은 흩어진 아이디어들을 한 곳에 모아 시너지를 낼 수 있는 '정점'이 되겠다는 의미를 담고 있습니다.

## ✨ 주요 기능

- **에디터 및 뷰어 모드 전환**
  - 글 작성에 집중할 수 있는 에디터 모드와 읽기 전용 뷰어 모드를 지원합니다.
- **메모 관리**
  - 폴더 생성, 즐겨찾기, 상단 고정, 휴지통 기능을 통해 메모를 분류하고 관리합니다.
- **UI/UX**
  - 다크 모드 및 라이트 모드를 지원합니다.
  - 글래스모피즘(Glassmorphism) 기반의 디자인을 적용했습니다.
  - 코드 블록 테마(딥 옵시디언, 글래스 캔버스, 미드나잇 인디고)를 변경할 수 있습니다.
- **데이터 백업 및 복원**
  - 메모와 폴더 구조를 JSON 파일로 내보내거나 가져올 수 있습니다.
- **에디터 기능**
  - Quill.js 기반의 리치 텍스트 에디터로 텍스트 포맷팅을 지원합니다.
  - 실시간 글자 수 확인 및 메모 검색 기능을 제공합니다.

## 🛠️ 기술 스택

- **Core:** HTML5, JavaScript (Vanilla)
- **Styling:** CSS3, Tailwind CSS (CDN)
- **Editor:** Quill.js
- **Syntax Highlighting:** Highlight.js
- **Icons:** Feather Icons

## 🚀 시작하기

1. 저장소를 클론(Clone)하거나 다운로드합니다.
2. 브라우저에서 `docs/index.html` 파일을 열면 별도의 로컬 서버 없이 바로 사용할 수 있습니다.
