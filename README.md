# Zenith (제니스)

**Zenith**는 당신의 아이디어와 기록이 도달할 수 있는 가장 높은 곳, 즉 '정점'을 의미합니다. 영감의 순간을 놓치지 않고 포착하여 체계적으로 관리할 수 있도록 돕는 프리미엄 웹 기반 메모 어플리케이션입니다.

## 🏔️ 이름의 기원 (Origin of Zenith)

Zenith(천정, 정점)\*\*라는 이름은 '모든 아이디어가 모여 최고의 가치를 발휘하는 곳'이라는 철학에서 탄생했습니다.
우리의 머릿속을 스쳐가는 수많은 생각들은 흩어지기 쉽지만, Zenith라는 하나의 꼭대기(정점)에 모였을 때 비로소 거대한 성과로 이어집니다. 사용자 경험의 정점, 생산성의 정점을 제공하겠다는 목표를 담고 있습니다.

## ✨ 주요 기능 (Features)

- **에디터 & 뷰어 모드 전환**
- 글 작성에 집중할 수 있는 **에디터 모드**와, 작성된 내용을 깔끔하게 읽을 수 있는 **뷰어 모드**를 원클릭으로 전환할 수 있습니다.
- **체계적인 메모 관리**
- 커스텀 폴더 생성, 즐겨찾기, 상단 고정, 휴지통 기능을 통해 수많은 메모를 효율적으로 분류하고 관리할 수 있습니다.
- **아름답고 섬세한 UI/UX**
- 다크 모드 및 라이트 모드 지원
- 부드러운 애니메이션과 글래스모피즘(Glassmorphism) 기반의 유려한 디자인
- 사용자 취향에 맞춘 코드 블록 테마(딥 옵시디언, 글래스 캔버스, 미드나잇 인디고) 제공
- **안전한 데이터 백업 & 복원**
- 작성된 모든 메모와 폴더 구조를 JSON 파일로 쉽게 내보내기(Export) 및 가져오기(Import) 할 수 있어 데이터를 안전하게 보존할 수 있습니다.
- **강력한 생산성 도구**
- 리치 텍스트 에디터(Quill.js) 탑재 및 주요 단축키 지원
- 실시간 하단 글자 수 카운팅
- 빠르고 즉각적인 메모 검색 기능

## 🛠️ 기술 스택 (Tech Stack)

- **Core:** HTML5, JavaScript (Vanilla)
- **Styling:** CSS3, Tailwind CSS (CDN)
- **Editor:** Quill.js
- **Syntax Highlighting:** Highlight.js
- **Icons:** Feather Icons

## 🚀 시작하기 (Getting Started)

1. 본 프로젝트 저장소를 클론(Clone) 하거나 다운로드합니다.
2. 브라우저에서 `docs/index.html` 파일을 열면 별도의 서버 설정 없이 즉시 어플리케이션을 사용할 수 있습니다.
